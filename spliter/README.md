# Spliter Folder

This directory contains utility scripts to benchmark different text splitting (chunking) strategies. Chunking is an essential step in RAG pipelines to ensure text stays within LLM context windows and retains local semantic context.

## 📁 Folder Contents

- **[textsplitter.py](file:///d:/Gen%20AI-project1/spliter/textsplitter.py)**: Demonstrates basic document splitting using `CharacterTextSplitter`. Reads local document templates and loops through output chunks.
- **`txt.txt`**: Sample source text file used to verify splitter properties.
